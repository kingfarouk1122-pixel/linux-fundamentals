# my linux journey

## I learned some basic linux command lines.

for example:

-ls to list the content of a directory

-pwd to show what is the current directory you're in right now.

-whoami to know who am I as a user

-cd to change my current directory.

-sudo apt install/update/upgrade to install something or update the system or upgrade anything that needs to be upgraded.

-<a command you don't know how to use> --help to help you know how to use this command.
